About the Demo

Oracle JET FixItFast is a field service demo application. This application shows several common mobile UX patterns used in phone-based applications. Demonstrated patterns include sign in, navigation drawer, dashboard, list view features, profiles, search, activity feeds and more.

Device services such as camera, dialer, email and more are made available. Selected animations are used for transitions, component effects and graphic fade animations. This demo is available for different mobile themes (Android and iOS) and for a web theme.

To learn more visit Oracle JET website.
